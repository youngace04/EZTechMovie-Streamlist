## Overview
StreamList is a React-based web application that allows users to create and manage a personalized streaming list. This project demonstrates the use of React Router for navigation, state management with React hooks, and user-friendly UI enhancements with CSS and icons.



## Features
- Navigation Bar with links to Home, Movies, Cart, and About pages.
- Add Items: Users can add movies or shows to their list.
- Edit Items: Update an existing item in the list.
- Delete Items: Remove items from the list.
- Mark Complete: Mark items as completed with visual feedback.
- Icons: Integrated using https://react-icons.github.io/react-icons/.
- Responsive Design: Styled with CSS for a clean and user-friendly interface.



## Technologies Used
- React
- React Router DOM
- React Icons
- CSS
